# FW AI Mockup v2

A modern ERP solution mockup showcasing enhanced UI layouts and AI interactivity.

**Live Demo:** [https://fwks-ai-demo.vercel.app/](https://fwks-ai-demo.vercel.app/)

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Headless UI** - Accessible component primitives
- **Heroicons** - Beautiful hand-crafted SVG icons

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/components` - Reusable Tailwind UI components
- `/sample` - Sample screenshots and HTML from existing ERP

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Import your repository at [vercel.com](https://vercel.com)
3. Vercel will automatically detect Next.js and configure the build

Or use the Vercel CLI:

```bash
npm install -g vercel
vercel
```

## Development

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
npm run lint     # Run ESLint
```
